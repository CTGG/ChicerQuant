    嗨，朋友，准备好接受数学的洗礼了嘛？
    让我们从股票的收益率开始跟随数学的脚步吧！
    收益率的计算大有玄机，需要考量以下两个方面：
    1、印花税
    2、复利

    印花税 是股票交易时需要向监管部门缴纳的税款，我国暂行的规定是卖出时征收千分之一
    单期收益率为R，则其对等的连续复利收益率应为(1＋R)的自然对数，即ln(1＋R)。
    之所以选择连续复利收益率，主要有一下两个原因：
    1、连续复利的公式以e为底数，可积可导指数可加，计算方便
    2、此处的模型抽象并不太影响现实结论
    3、它的概率分布较接近于正态分布
    接近正态分布又有什么好处呢？请接着探寻.
